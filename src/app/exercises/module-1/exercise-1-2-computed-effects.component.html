Key Concepts

  • computed(() => ...): Creates a derived signal that automatically recalculates when dependencies change
  • effect(() => ...): Runs side effects when signals change (logging, API calls, etc.)
  • Computed signals are lazy: They only recalculate when accessed and dependencies change
  • Effects run automatically: They execute whenever their dependencies change
  • Computed signals are memoized: They cache results until dependencies change
