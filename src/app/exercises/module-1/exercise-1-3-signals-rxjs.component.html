When to Use What?

Use Case Signals RxJS
Simple state ✅ Best ❌ Overkill
Derived values ✅ computed() ⚠️ map(), combineLatest()
Async operations ❌ Limited ✅ Excellent
Stream transformations ❌ No ✅ Excellent
Side effects ⚠️ effect() ✅ subscribe()
HTTP requests ❌ Use toSignal() ✅ HttpClient

Key Concepts

  • toSignal(observable): Converts an Observable to a Signal
  • toObservable(signal): Converts a Signal to an Observable
  • Signals are best for: Simple state, derived values, synchronous operations
  • RxJS is best for: Async operations, stream transformations, complex data flows
  • Use both together: Convert between them as needed for the best of both worlds
