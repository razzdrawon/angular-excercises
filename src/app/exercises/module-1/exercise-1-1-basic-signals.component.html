Key Concepts

  • signal(value): Creates a signal with an initial value
  • signal(): Reads the current value of a signal
  • signal.set(value): Sets a new value
  • signal.update(fn): Updates value using a function
  • computed(() => ...): Creates a derived signal that automatically updates when dependencies change
