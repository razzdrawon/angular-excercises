## Curso Angular Avanzado - Plan de Estudio

### Estructura del Curso

#### Módulo 0: Recapitulación Conceptual (Conversacional)
- Signals: what/why/advantages vs RxJS
- Standalone components: evolution, benefits, migration
- New control flow (`@if`, `@for`, `@switch`)
- Performance: change detection, OnPush, modern optimizations
- Architecture: patterns, state management, composition
- Forms: template vs reactive, advanced validation
- Routing: guards, resolvers, lazy loading
- Testing: strategies, tools, best practices
- Advanced features: SSR, PWA, micro-frontends
- Angular 21: new features, trends, future

#### Módulo 1: Angular Moderno - Signals y Reactividad
- Advanced signals (`computed`, `effect`, batching)
- Signals vs RxJS, integration points
- `toSignal`, `toObservable`, hybrid patterns
- Exercise: reactive dashboard with signals

#### Módulo 2: Standalone Components y Control Flow
- Standalone architecture patterns
- New control flow syntax and best practices
- Dependency injection without NgModules
- Exercise: standalone component library

#### Módulo 3: Performance y Optimización
- Change detection strategies
- OnPush + signals for high performance
- Lazy loading, code splitting
- Virtual scrolling, `trackBy`
- Exercise: 10k+ item optimized list

#### Módulo 4: Arquitectura Avanzada
- State management with signals/services
- Feature organization strategies
- Composable architecture patterns
- Custom directives/pipes
- Exercise: permissions and roles system

#### Módulo 5: Forms Avanzados y Validación
- Advanced reactive forms
- Custom + async validators
- Dynamic form arrays/groups
- Signals plus forms
- Exercise: complex multi-step form

#### Módulo 6: Routing Avanzado
- Guards, resolvers, standalone lazy loading
- Route data and params with signals
- Preloading strategies
- Exercise: multi-tenant routing app

#### Módulo 7: Testing Avanzado
- Unit testing signals/components
- Component + integration testing
- E2E testing flows
- Exercise: full testing suite

#### Módulo 8: Features Avanzadas
- Angular Elements, SSR, hydration
- PWAs, offline strategies
- Micro-frontends
- Exercise: SSR-enabled PWA integration

### Metodología
- Each module blends concise theory with independent hands-on exercises
- Every exercise ships as a standalone component
- A final integration project combines all exercises
- Code reviews and best practices baked into each module
- Real-time Q&A during practice sessions

### Estructura de Archivos
- `src/app/exercises/` – independent exercises per module (`exercise-1-1`, etc.)
- `src/app/integration/` – final integration project
- `src/app/routes/` – navigation between exercises
- Each exercise can run standalone or inside the integrated project
